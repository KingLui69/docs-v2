CoordMode

Bestimmt den Koordinaten-Modus für verschiedene Befehle bzgl. relativer oder absoluter Fenster-/Bildschirmposition.

CoordMode, ToolTip|Pixel|Mouse|Caret|Menu [, Screen|Window|Client]

Parameter

Param1

ToolTip: Beeinflusst ToolTip.

Pixel: Beeinflusst PixelGetColor, PixelSearch und ImageSearch.

Mouse: Beeinflusst MouseGetPos, Click und MouseMove/Click/Drag.

Caret: Beeinflusst die internen Variablen A_CaretX und A_CaretY.

Menu: Beeinflusst den Befehl Menu Show, falls dessen Koordinaten angegeben worden sind.

Param2

Wenn Param2 nicht vorhanden ist, wird Screen standardmäßig verwendet.

Screen: Koordinaten beziehen sich auf den Desktop (ganzer Bildschirm).

Relative: Koordinaten beziehen sich auf das aktive Fenster.

Window: Synonymous with Relative and recommended for clarity.

Client: Koordinaten beziehen sich auf den Client-Bereich des aktiven Fensters (ohne Titelleiste, Standardmenü und Rahmen). Client-Koordinaten sind weniger abhängig von Version und Design des Betriebssystems.

Bemerkungen

If this command is not used, all commands except those documented otherwise (e.g. WinMove and InputBox) use coordinates that are relative to the active window's client area.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann man ändern, wenn dieser Befehl bereits im automatischen Ausführungsbereich (am Anfang des Scripts) definiert wird.

Siehe auch

Click, MouseMove, MouseClick, MouseClickDrag, MouseGetPos, PixelGetColor, PixelSearch, ToolTip, Menu

Beispiel

CoordMode, ToolTip, Screen  ; Platziert ToolTips auf absolute Bildschirmkoordinaten:
CoordMode, ToolTip  ; Gleicher Effekt wie oben, da "Screen" die Standardeinstellung ist.
