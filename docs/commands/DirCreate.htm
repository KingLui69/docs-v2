DirCreate

Erstellt ein(en) Verzeichnis/Ordner. 

DirCreate, DirName

Parameter

Verzeichnisname

Name des Verzeichnisses, das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Dieser Befehl erstellt auch alle übergeordneten Verzeichnisse, die in Verzeichnisname vorkommen, falls sie noch nicht vorhanden sind.

Siehe auch

DirDelete

Beispiel

DirCreate, C:\Test1\My Images\Folder2
