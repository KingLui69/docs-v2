Deref [v2]

Expands variable references and escape sequences contained inside other variables.

Deref, OutputVar, String

Parameter

AusgabeVar

Name der Variable, in der das Ergebnis gespeichert werden soll.

String

A string containing variable references and/or escape sequences.

Bemerkungen

Jeder schlecht formatierte Variablenverweis wird im erweiterten Ergebnis weggelassen. The same is true if OutputVar is expanded into itself; in other words, any references to OutputVar inside String's variables will be omitted from the expansion (note however that String itself can be %OutputVar%).

Within a function, each variable reference in String may resolve to a local or global variable. Standard variable scope rules apply.

Beispiele

var1 := "test"
var2 := Chr(37) "var1"  ; Chr(37) is %
Deref OutputVar, %var2%
MsgBox %OutputVar%  ; Shows "test"
