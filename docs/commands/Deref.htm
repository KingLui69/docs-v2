Deref [v2]

Erweitert Variablenverweise und Escapesequenzen, die sich in anderen Variablen befinden.

Deref, AusgabeVar, String

Parameter

AusgabeVar

Name der Variable, in der das Ergebnis gespeichert werden soll.

String

Ein String mit Variablenverweisen und/oder Escape-Sequenzen.

Bemerkungen

Jeder schlecht formatierte Variablenverweis wird im erweiterten Ergebnis weggelassen. Das gleiche gilt auch, wenn sich AusgabeVar in sich selbst erweitert. Das heißt, dass Verweise auf AusgabeVar innerhalb der Variablen von String beim Erweitern weggelassen werden (allerdings kann String selbst %AusgabeVar% sein).

Innerhalb einer Funktion wird jede Variable in String immer in eine lokale oder globale Variable aufgelöst. Die Standard-Regeln bzgl. der Gültigkeitsbereiche von Variablen gelten auch hier.

Beispiele

var1 := "test"
var2 := Chr(37) "var1"  ; Chr(37) ist %
Deref AusgabeVar, %var2%
MsgBox %AusgabeVar%  ; Zeigt "test" an
