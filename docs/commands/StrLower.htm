StrLower / StrUpper

Wandelt den String in Klein- oder Großbuchstaben um.

StrLower, AusgabeVar, String [, T]
StrUpper, AusgabeVar, String [, T]

Parameter

AusgabeVar

Name der Variable, in der der neu umgewandelte String gespeichert werden soll.

String

Der String, der umgewandelt werden soll.

T

Sobald dieser Parameter der Buchstabe T ist, wird der String in ein Titel umgewandelt. Zum Beispiel wird "VOM Winde VERWEHT" zu "Vom Winde Verweht".

Bemerkungen

Mit RegExMatch kann überprüft werden, ob ein Zeichen oder String vollständig groß- oder kleingeschrieben ist. Zum Beispiel:

var := "abc"
if RegExMatch(var, "^[a-z]$")
    MsgBox var besteht vollkommen aus Kleinbuchstaben.
if !RegExMatch(var, "[A-Z]")
    MsgBox var enthält keinen einzigen Großbuchstaben.
