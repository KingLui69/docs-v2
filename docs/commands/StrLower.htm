Parameter

AusgabeVar

The name of the variable in which to store the newly converted string.

String

The string to convert.

T

Sobald dieser Parameter der Buchstabe T ist, wird der String in ein Titel umgewandelt. Zum Beispiel wird "VOM Winde VERWEHT" zu "Vom Winde Verweht".

Bemerkungen

To detect whether a character or string is entirely uppercase or lowercase, use RegExMatch. Zum Beispiel:

var := "abc"
if RegExMatch(var, "^[a-z]$")
    MsgBox var is entirely composed of lowercase characters.
if !RegExMatch(var, "[A-Z]")
    MsgBox var does not contain any uppercase characters.
