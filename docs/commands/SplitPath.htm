SplitPath

Teilt ein Dateiname oder eine URL in Name, Verzeichnis, Erweiterung und Laufwerk auf.

SplitPath, Path [, OutFileName, OutDir, OutExtension, OutNameNoExt, OutDrive]

Parameter

Path

The path or file name to be analyzed.

Dateiname

Name der Ausgabevariable, in der der Dateiname ohne Pfad gespeichert werden soll. Die Dateierweiterung ist inbegriffen.

Verzeichnis

Name der Ausgabevariable, in der das Verzeichnis der Datei, einschließlich Laufwerksbuchstabe oder Freigabename (falls vorhanden), gespeichert werden soll. Der letzte Backslash ist nicht enthalten, auch dann nicht, wenn sich die Datei im Wurzelverzeichnis des Laufwerks befindet.

Erweiterung

Name der Ausgabevariable, in der die Dateierweiterung gespeichert werden soll (z. B. TXT, DOC oder EXE). Der Punkt (.) ist nicht inbegriffen.

NameOhneErw

Name der Ausgabevariable, in der der Dateiname ohne Pfad, Punkt und Erweiterung gespeichert werden soll.

Laufwerk

Name der Ausgabevariable, in der der Laufwerksbuchstabe oder Servername der Datei gespeichert werden soll. Wenn sich die Datei auf einem lokalen oder zugeordneten Laufwerk befindet, wird die Variable den Laufwerksbuchstaben gefolgt von einem Doppelpunkt enthalten (ohne Backslash). Befindet sich die Datei auf einem Netzwerkpfad (UNC), wird die Variable den Freigabenamen enthalten (z. B. \\Workstation01).

Bemerkungen

Falls eine bestimmte Information nicht benötigt wird, kann die entsprechende Ausgabevariable weggelassen werden.

If Path contains a filename that lacks a drive letter (that is, it has no path or merely a relative path), OutDrive will be made blank but all the other output variables will be set correctly. Ist kein Pfad vorhanden, wird Verzeichnis leer gemacht; ist ein Pfad, aber kein Dateiname vorhanden, wird Dateiname und NameOhneErw leer gemacht.

Derzeitige Dateien und Verzeichnisse im Dateisystem werden nicht von diesem Befehl überprüft. It simply analyzes the provided string.

Platzhalter (* und ?) und andere Zeichen, die im Dateinamen nicht erlaubt sind, können in EingabeVar vorkommen. Das gilt nicht für Doppelpunkt, Backslash und Punkt, da diese als Abgrenzung für Laufwerksbuchstaben, Verzeichnisse und Erweiterungen dienen.

Unterstützung für URLs: If Path contains a colon-double-slash, such as http://domain.com or ftp://domain.com, OutDir is set to the protocol prefix + domain name + directory (e.g. http://domain.com/images) and OutDrive is set to the protocol prefix + domain name (e.g. http://domain.com). Alle anderen Ausgabevariablen werden in Bezug auf ihren oben genannten Definitionen gesetzt.

Beispiel

KompletterDateiname = C:\Eigene Dokumente\Adressliste.txt

; Um nur den Dateinamen ohne Pfad abzurufen:
SplitPath, KompletterDateiname, Dateiname

; Um nur das Verzeichnis abzurufen:
SplitPath, KompletterDateiname,, Verzeichnis

; Um alle Informationen abzurufen:
SplitPath, KompletterDateiname, Name, Verz, Erw, NameOhneErw, Laufwerk

; Die oben genannten Variablen werden wie folgt gesetzt:
; Name = Adressliste.txt
; Verz = C:\Eigene Dokumente
; Erw = txt
; NameOhneErw = Adressliste
; Laufwerk = C:
