ListLines

Aktiviert oder Deaktiviert die Zeilen-Protokollierung, oder zeigt die aktuell ausgeführten Script-Zeilen an.

ListLines [, On|Off]

Parameter

On|Off

Ist dieser Parameter leer oder nicht vorhanden, wird der Verlauf der aktuell ausgeführten Zeilen angezeigt. Der erste Parameter wirkt sich wie folgt nur auf das Verhalten des aktuellen Threads aus:

On: Alle danach ausgeführten Zeilen werden im Verlauf eingefügt.

Off: Alle danach ausgeführten Zeilen werden im Verlauf weggelassen. Das ist der Standardwert für alle Scripts.

Bemerkungen

ListLines (ohne Parameter) ist das gleiche wie, als würde man den Menüpunkt "View->Lines most recently executed" im Hauptfenster auswählen. Dieser Befehl kann dabei helfen, ein Script zu debuggen. Da jedoch die Zeilen-Protokollierung standardmäßig deaktiviert ist, muss "ListLines On" verwendet werden, bevor der zu debuggende Codebereich ausgeführt wird.

Mit ListLines Off/On können bestimmte Zeilen im Verlauf weggelassen werden, um zu verhindern, dass sich der Verlauf zu schnell füllt (z. B. bei einer Schleife mit schnellen Iterationen). Zusätzlich ist die Performance um einige Prozent reduziert, während die Zeilen-Protokollierung aktiviert ist.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitgesteuerte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann man ändern, wenn dieser Befehl bereits im automatischen Ausführungsbereich (am Anfang des Scripts) definiert wird.

Obwohl die integrierte Variable "A_ListLines" nicht verfügbar ist, kann man eine ähnliche Funktionalität erreichen, indem man den folgenden Code in das Script einfügt:

ListLines(PassTrueToTurnOnOrFalseToTurnOff)  ; Gibt die vorherige Einstellung von ListLines zurück (vor diesem Aufruf).
{
    static sListLines := true  ; Die Standardeinstellung für alle Scripts ist "ListLines On".
    ListLines % PassTrueToTurnOnOrFalseToTurnOff ? "On" : "Off"  ; Führt ListLines bedingungslos aus, um danach ausgeführte Zeilen im Verlauf wegzulassen.
    ListLines_prev := sListLines
    sListLines := PassTrueToTurnOnOrFalseToTurnOff
    Return ListLines_prev
}

; Um die obere Funktion anzuwenden:
prev_ListLines := ListLines(false)  ; ListLines wird vorübergehend deaktiviert.
; ...
ListLines(prev_ListLines)  ; Die vorherige Einstellung von ListLines wird wiederhergestellt.

Beachtet zudem, dass die integrierten Variablen A_LineNumber und A_LineFile die derzeit ausgeführte Zeilennummer und den dazugehörigen Dateinamen enthalten.
