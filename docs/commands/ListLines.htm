ListLines

Enables or disables line logging or displays the script lines most recently executed.

ListLines [, On|Off]

Parameter

On|Off

Ist dieser Parameter leer oder nicht vorhanden, wird der Verlauf der aktuell ausgeführten Zeilen angezeigt. The first parameter affects only the behavior of the current thread as follows:

On: Alle danach ausgeführten Zeilen werden im Verlauf eingefügt.

Off: Alle danach ausgeführten Zeilen werden im Verlauf weggelassen. Das ist der Standardwert für alle Scripts.

Bemerkungen

ListLines (ohne Parameter) ist das gleiche wie, als würde man den Menüpunkt "View->Lines most recently executed" im Hauptfenster auswählen. Dieser Befehl kann dabei helfen, ein Script zu debuggen. However, since line logging is disabled by default, "ListLines On" must be used before executing the section of code which is to be debugged.

Mit ListLines Off/On können bestimmte Zeilen im Verlauf weggelassen werden, um zu verhindern, dass sich der Verlauf zu schnell füllt (z. B. bei einer Schleife mit schnellen Iterationen). Additionally, performance is reduced by a few percent while line logging is enabled.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann man ändern, wenn dieser Befehl bereits im automatischen Ausführungsbereich (am Anfang des Scripts) definiert wird.

Obwohl die interne Variable "A_ListLines" nicht existiert, kann dennoch eine ähnliche Funktionalität erreicht werden, sobald das folgende Beispiel im Script eingefügt wird:

ListLines(PassTrueToTurnOnOrFalseToTurnOff)  ; Gibt die vorherige Einstellung von ListLines zurück (vor diesem Aufruf).
{
    static sListLines := true  ; Die Standardeinstellung für alle Scripts ist "ListLines On".
    ListLines % PassTrueToTurnOnOrFalseToTurnOff ? "On" : "Off"  ; Führt ListLines bedingungslos aus, um danach ausgeführte Zeilen im Verlauf wegzulassen.
    ListLines_prev := sListLines
    sListLines := PassTrueToTurnOnOrFalseToTurnOff
    Return ListLines_prev
}

; Um die obere Funktion anzuwenden:
prev_ListLines := ListLines(false)  ; ListLines wird vorübergehend deaktiviert.
; ...
ListLines(prev_ListLines)  ; Die vorherige Einstellung von ListLines wird wiederhergestellt.

Beachtet zudem, dass die internen Variablen A_LineNumber und A_LineFile die derzeit ausgeführte Zeilennummer und den dazugehörigen Dateinamen enthalten.
