FileDelete

Löscht eine oder mehrere Dateien. 

FileDelete, Dateimuster

Parameter

Dateimuster

Name einer Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

To remove an entire folder, along with all its sub-folders and files, use DirDelete.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

Bei Problemen enthält ErrorLevel die Anzahl an Dateien, die nicht gelöscht werden konnten, ansonsten wird ErrorLevel auf 0 gesetzt. Das Löschen einer musterbasierten Suche wie *.tmp wird als Erfolg angesehen, auch dann, wenn keine Dateien gefunden wurden; folglich wird ErrorLevel auf 0 gesetzt.

Falls Dateien gefunden wurden, wird A_LastError auf 0 gesetzt, oder auf den aktuellsten Rückgabewert der Systemfunktion GetLastError(). Ansonsten enthält A_LastError einen Fehlercode, der darauf hinweist, warum keine Dateien gefunden wurden.

Bemerkungen

Eine schreibgeschützte Datei kann nur gelöscht werden, wenn ihr vorher der Schreibschutzattribut entfernt wird. Zum Beispiel: FileSetAttrib, -R, C:\My File.txt.

Siehe auch

FileRecycle, DirDelete, FileCopy, FileMove

Beispiel

FileDelete, C:\Temp-Dateien\*.tmp
