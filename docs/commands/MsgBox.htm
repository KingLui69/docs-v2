MsgBox

Zeigt den angegebenen Text in einem kleinen Fenster an, das eine oder mehrere Schaltflächen enthält (z. B. Ja und Nein).

MsgBox, Text
MsgBox [, Optionen, Titel, Text, Zeitüberschreitung]

Parameter

Text

If all the parameters are omitted, the MsgBox will display the text "Press OK to continue." Otherwise, this parameter is the text displayed inside the message box to instruct the user what to do, or to present information.

Escapesequenzen können verwendet werden, um besondere Zeichen zu kennzeichnen. Zum Beispiel kennzeichnet `n ein Zeilenumbruchszeichen, das die aktuelle Zeile beendet und eine neue beginnt. Folglich würde mit Text1`n`nText2 eine leere Zeile zwischen Text1 und Text2 erzeugt.

Falls der Text zu lang ist, kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um ihn besser lesbar und verwaltbar zu machen.

Optionen

Indicates the type of message box and the possible button combinations.  If blank or omitted, it defaults to 0. Siehe untere Tabelle für zulässige Werte.

When command syntax is used, this parameter must be either a literal number or a forced expression such as % Options. Andere nicht-leere Werte werden nicht als diesen Parameter erkannt, sondern als Text im Einzelparametermodus.

Titel

Der Titel der MsgBox. Ist dieser Parameter leer oder nicht vorhanden, wird standardmäßig der aktuelle Name des Scripts verwendet (ohne Pfad).

Zeitüberschreitung

(optional) Timeout in seconds (can contain a decimal point).  If this value exceeds 2147483 (24.8 days), it will be set to 2147483.  After the timeout has elapsed the message box will be automatically closed and A_MsgBoxResult will contain the value "Timeout".

Bekannte Einschränkung: If the MsgBox contains only an OK button, A_MsgBoxResult will indicate that the OK button was pressed if the MsgBox times out while its own thread is interrupted by another.

Ergebnis

The built-in variable A_MsgBoxResult returns one of the following strings to represent which button the user pressed in the most recent MsgBox command:

OK
Cancel (Abbrechen)
Ja
No (Nein)
Abort (Abbrechen)
Retry (Wiederholen)
Ignore (Ignorieren)
TryAgain
Continue
Timeout (that is, the word "timeout" is present if the MsgBox timed out)

Bemerkungen

Die Werte in der oberen Tabelle können addiert werden, um mehrere Eigenschaften zu kombinieren. Zum Beispiel bewirkt die Angabe von 260 (256 + 4) in den Optionen, dass die Schaltflächen Ja und Nein in der MsgBox angezeigt werden und Nein zur Standard-Schaltfläche gemacht wird. In Hexadezimal wäre das 0x104 (0x100 + 0x4).

MsgBox hat eine intelligente Komma-Handhabung, daher müssen Kommas im Text-Parameter nicht mit einem Escapezeichen versehen werden.

To determine which button the user pressed in the most recent MsgBox, use the A_MsgBoxResult variable. Zum Beispiel:

MsgBox, 4,, Möchten Sie fortfahren? (press Yes or No)
if A_MsgBoxResult = "Yes"
    MsgBox You pressed Yes.
else
    MsgBox You pressed No.

Die Namen der Schaltflächen können mit diesem Beispiel geändert werden.

Tipp: Der Text einer aktiven MsgBox kann mit STRG+C in die Zwischenablage kopiert werden. Das gilt für alle Fenster dieser Art, nicht nur für solche, die von AutoHotkey erstellt wurden.

MsgBox mit einem GUI-Fenster verwenden: Ein GUI-Fenster kann eine modale MsgBox mithilfe von Gui, +OwnDialogs darstellen. A modal MsgBox prevents the user from interacting with the GUI window until the MsgBox is dismissed. In einem solchen Fall ist es nicht notwendig, System Modal oder Task Modal in den Optionen anzugeben.

Wenn Gui +OwnDialogs nicht aktiv ist, kann die Option Task Modal (8192) dazu verwendet werden, alle Fenster des Scripts zu deaktivieren, solange die MsgBox offen ist.

Die Schaltfläche Hilfe: Sobald die entsprechende Option (83) in den Optionen vorhanden ist, wird eine Hilfe-Schaltfläche angezeigt, die aber nur funktioniert, wenn die beiden folgenden Bedingungen erfüllt sind:

  1. Die MsgBox wurde einem GUI-Fenster mittels Gui +OwnDialogs untergeordnet.
  2. Das Script überwacht die WM_HELP-Nachricht (0x53). Zum Beispiel: OnMessage(0x53, "WM_HELP"). Beim Aufrufen der Funktion WM_HELP() kann zum Beispiel ein weiteres Fenster angezeigt werden, um den Benutzer zusätzliche Hinweise mitzuteilen.

Der Schließen-Button (in der Titelleiste der MsgBox): Since the MsgBox window is a built-in feature of the operating system, its X button is enabled only when certain buttons are present. Gibt es nur einen OK-Button, ist das Anklicken von X das gleiche wie, als würde man OK drücken. Ansonsten ist der X-Button nur verfügbar, wenn ein Abbrechen-Button vorhanden ist. In diesem Fall ist das Anklicken von X das gleiche wie, als würde man Abbrechen drücken.

Beispiel

Function Syntax

MsgBox("This is the 1-parameter method. If only one parameter is passed, the first parameter becomes the message.")

MsgBox(64, "3-parameter method", "The first parameter determins the type of message box.`n" 
	. "The second parameter becomes the window title.`n"
	. "The third parameter is displayed as the message.")

MsgBox(4, "", "Do you want to continue? (Press YES or NO)")
if A_MsgBoxResult = "No"
    return
	
MsgBox(4, "4-parameter method", "this MsgBox will time out in 5 seconds.  Continue?", 5)
if A_MsgBoxResult = "Timeout"
    MsgBox("You didn't press YES or NO within the 5-second period.")
else if A_MsgBoxResult = "No"
    return
; Include a variable and perform a math calculation in the message.
var := 10
Msgbox("The initial value is: " var)
MsgBox("The result is: " var * 2)

Command Syntax

MsgBox Das ist die 1-Parameter-Methode. Commas (,) do not need to be escaped.

MsgBox, 64
	, 3-parameter method
	, This is the 3-parameter method. Commas (,) do not need to be escaped.
	
MsgBox, 4, , Do you want to continue? (Press YES or NO)
if A_MsgBoxResult = "No"
    return
	
MsgBox, 4,,  4-parameter method: Diese MsgBox wird in 5 Sekunden automatisch geschlossen.  Continue?, 5
if A_MsgBoxResult = "Timeout"
    MsgBox You didn't press YES or NO within the 5-second period.
else if A_MsgBoxResult = "No"
    return
;Include a variable in the message.
var := 10
MsgBox The initial value is: %var	;The preciding % enables to yield the contents of the variable.
MsgBox % "The result is: " var * 2	;By preceding any parameter with "% ", it becomes an expression.
