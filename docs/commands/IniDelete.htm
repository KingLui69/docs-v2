IniDelete

Entfernt einen Wert aus einer Standard-INI-Datei. 

IniDelete, Dateiname, Sektion [, Schlüssel]

Parameter

Dateiname

Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

Sektion

Name der Sektion aus der INI-Datei. Solche Sektionen werden mit eckigen Klammern dargestellt (die Klammern dürfen nicht in diesem Parameter stehen).

Schlüssel

Name des Schlüssels aus der INI-Datei. Ist dieser Parameter nicht vorhanden, wird die ganze Sektion gelöscht.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Eine typische INI-Datei sieht wie folgt aus:

[Sektionsname]
Schlüssel=Wert

Siehe auch

IniRead, IniWrite, RegDelete

Beispiel

IniDelete, C:\Temp\EigeneDatei.ini, Sektion2, Schlüssel
