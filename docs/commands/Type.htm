Type

Gibt den exakten Typ eines Wert zurück.

AusgabeVar := Type(Wert)

Rückgabewert

Der Rückgabewert ist ein String, der den Typ von Wert kennzeichnet. Mögliche Werte sind:

  • String
  • Integer
  • Float
  • Object

Wenn Wert ein Objekt ist, könnte der Rückgabewert "Object" sein, oder ein integrierter Typ mit der genauen Bezeichnung wie "Func", "FileObject" oder "ComObject".

Bemerkungen

Diese Funktion sollte man normalerweise nicht verwenden, um zu überprüfen, ob ein Wert numerisch ist, weil numerische Strings in mathematischen Ausdrücken und bei integrierten Funktionen gültig sind. In einigen Fällen jedoch ist der genaue Typ eines Wertes wichtiger. Wenn man NumPut beispielsweise eine Variable mit einem reinen Integer (kein numerischer String) übergibt, wird der Integer anstelle der Variablenadresse verwendet.

Wenn überprüft werden soll, ob ein Wert als Zahl verwendet werden kann, verwende den Ausdruck Wert is Typ (Typ ist "number", "integer" oder "float").

Wenn überprüft werden soll, ob der angegebene Wert ein Objekt beliebigen Typs ist, verwende den Ausdruck Wert is 'object'.

Wenn überprüft werden soll, ob ein Objekt von einer bestimmten benutzerdefinierten Klasse abgeleitet ist, verwende Wert is Klassenname.

Beispiele

a := 1, b := 2.0, c := "3"
MsgBox % Type(a)  ; Integer
MsgBox % Type(b)  ; Float
MsgBox % Type(c)  ; String
