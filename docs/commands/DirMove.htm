DirMove

Verschiebt einen Ordner, einschließlich seine Unterordner und Dateien. Kann einen Ordner auch umbenennen.

DirMove, Source, Dest [, Flag]

Parameter

Quelle

Name des Quellverzeichnisses (ohne Backslash), das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Zum Beispiel: C:\Mein Ordner

Ziel

Der neue Pfad und Name des Verzeichnisses (ohne Backslash), das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Zum Beispiel: D:\Eigener Ordner. Hinweis: Ziel ist der aktuelle Pfad und Name, das das Verzeichnis wird, nachdem es verschoben wurde; es ist nicht das Verzeichnis, in dem Quelle verschoben worden ist (außer bei der unten erwähnten Einschränkung).

Flag

(Optionen) Verwendet eins der folgenden Zeichen:

0 (Standard): Vorhandene Dateien werden nicht überschrieben. Die Operation wird fehlschlagen, wenn Ziel bereits als Datei oder Verzeichnis existiert.

1: Vorhandene Dateien werden überschrieben. However, any files or subfolders inside Dest that do not have a counterpart in Source will not be deleted. Known limitation: If Dest already exists as a folder and it is on the same volume as Source, Source will be moved into it rather than overwriting it. Das kann mit der nächsten Option verhindert werden.

2: Das gleiche wie in Modus 1, aber ohne Einschränkung.

R: Bewirkt, dass das Verzeichnis umbenannt anstatt verschoben wird. Obwohl Umbenennen normalerweise das Gleiche wie Verschieben ist, ist es in Fällen wie "Alles oder nichts" hilfreich. Das heißt, um zu verhindern, dass die Operation nur teilweise erfolgreich ist, wenn Quelle oder eine ihrer Dateien gesperrt ist (in Benutzung). Obwohl diese Methode nicht in der Lage ist, Quelle auf einen anderen Datenträger zu verschieben, kann sie damit dennoch in ein anderes Verzeichnis auf demselben Datenträger verschoben werden. Die Operation wird fehlschlagen, wenn Ziel bereits als Datei oder Verzeichnis vorkommt.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

DirMove moves a single folder to a new location. Um stattdessen den Inhalt eines Ordners zu verschieben (alle Dateien und Unterordner), sieht die Beispiele bei FileMove.

Wenn sich Quelle und Ziel auf unterschiedlichen Datenträgern oder UNC-Pfade befinden, erfolgt Kopieren/Löschen, anstatt Verschieben.

Siehe auch

DirCopy, FileCopy, FileMove, FileDelete, File-loops, DirSelect, SplitPath

Beispiel

DirMove, C:\My Folder, D:\My Folder  ; Move to a new drive. 
DirMove, C:\My Folder, C:\My Folder (renamed), R  ; Simple rename.
DirMove, C:\My Folder, C:\New Location\My Folder, R  ; Folders can be "renamed into" another location as long as it's on the same volume.
