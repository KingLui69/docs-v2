StatusBarGetText

Ermittelt den Text einer Standard-Statusleiste. 

StatusBarGetText, AusgabeVar [, Bereichsnummer, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

AusgabeVar

Name der Variable, in der der ermittelte Text gespeichert werden soll.

Bereichsnummer

Which part number of the bar to retrieve. Ist dieser Parameter leer oder nicht vorhanden, gilt standardmäßig 1, das üblicherweise der Bereich ist, der am interessantesten ist.

Fenstertitel

Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.

Fenstertext

In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (abrufbar mit Windows Spy). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.

Titelausnahme

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt. Bei Problemen wird AusgabeVar zudem leer gemacht.

Bemerkungen

Dieser Befehl versucht, die erste Standard-Statusleiste eines Fensters zu lesen (typisches Steuerelement von Microsoft: msctls_statusbar32).  Some programs use their own status bars or special versions of the MS common control, in which case the text cannot be retrieved.

Rather than using this command in a loop, it is usually more efficient to use StatusBarWait, which contains optimizations that avoid the overhead of repeated calls to StatusBarGetText.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.
