WinGetText

Ermittelt den Text des angegebenen Fensters.

WinGetText, AusgabeVar [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

AusgabeVar

Name der Variable, in der der ermittelte Text gespeichert werden soll.

Fenstertitel

Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.

Fenstertext

In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (abrufbar mit Windows Spy). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.

Titelausnahme

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Der ermittelte Text ist generell das gleiche, was Windows Spy für dieses Fenster anzeigt. Wenn DetectHiddenText deaktiviert ist, werden versteckte Texte in AusgabeVar weggelassen.

Jedes Textelement endet mit einem CR- und LF-Zeilenumbruchszeichen, die im Script als `r`n dargestellt werden können. To extract individual lines or substrings, use functions such as InStr() and SubStr(). Eine Parsing-Schleife kann auch verwendet werden, um Zeilen oder Worte einzeln zu überprüfen.

Falls der ermittelte Text gekürzt (unvollständig) erscheint, kann VarSetCapacity(AusgabeVar, 55) vor WinGetText ausprobiert werden [ersetze 55 mit einer Zahl, die wesentlich länger als der gekürzte Text ist]. Das ist notwendig, weil einige Anwendungen nicht richtig auf die WM_GETTEXTLENGTH-Nachricht reagieren, wodurch AutoHotkey AusgabeVar zu klein macht, so dass der ganzen Text nicht mehr reinpasst.

This command might use a large amount of RAM if the target window (e.g. an editor with a large document open) contains a large quantity of text. To avoid this, it might be possible to retrieve only portions of the window's text by using ControlGetText instead. In any case, a variable's memory can be freed later by assigning it to nothing, i.e. OutputVar := ""

Um eine Liste mit allen Steuerelementen eines Fensters abzurufen, befolge dieses Beispiel: WinGet, OutputVar, ControlList, WinTitle

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.
