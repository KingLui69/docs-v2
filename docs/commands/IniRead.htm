IniRead

Liest einen Wert aus einer normalen INI-Datei.

IniRead, AusgabeVar, Dateiname [, Sektion, Schlüssel, Standardwert]

Parameter

AusgabeVar

Name der Variable, in der der ermittelte Wert gespeichert werden soll. Falls der Wert nicht ermittelbar ist, enthält die Variable den Standardwert (siehe unten).

Sobald der Schlüssel weggelassen wird, wird der Inhalt der angegebenen Sektion abgerufen, ohne jegliche Kommentare.

Wenn sowohl Schlüssel als auch Sektion weggelassen werden, enthält AusgabeVar eine Liste mit Sektionsnamen, die durch LF-Zeichen (`n) getrennt sind.

Dateiname

Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

Sektion

Name der Sektion aus der INI-Datei. Solche Sektionen werden mit eckigen Klammern dargestellt (die Klammern dürfen nicht in diesem Parameter stehen).

Schlüssel

Name des Schlüssels aus der INI-Datei.

Standardwert

Dieser Wert wird in AusgabeVar gespeichert, wenn der angeforderte Schlüssel nicht gefunden wird. Ist dieser Parameter nicht vorhanden, gilt ein leerer String als Standardwert.

Dieser Parameter entfällt, sobald der Schlüssel weggelassen wird.

ErrorLevel

Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Das Betriebssystem entfernt automatisch führende oder nachfolgende Leer-/Tabulatorzeichen in AusgabeVar.

Werte über 65.535 Zeichen verursachen wahrscheinlich widersprüchliche Ergebnisse.

Eine typische INI-Datei sieht wie folgt aus:

[Sektionsname]
Schlüssel=Wert

Unicode: IniRead und IniWrite sind auf die externen Funktionen GetPrivateProfileString und WritePrivateProfileString angewiesen, um Werte lesen oder schreiben zu können. Diese Funktionen unterstützen Unicode nur in UTF-16-Dateien; alle anderen Dateien verwenden den Standard-ANSI-Zeichensatz des Systems.

Siehe auch

IniDelete, IniWrite, RegRead, dateilesende Schleife, FileRead

Beispiel

IniRead, AusgabeVar, C:\Temp\EigeneDatei.ini, Sektion2, Schlüssel
MsgBox, Der Wert ist %AusgabeVar%.
