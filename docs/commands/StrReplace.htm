StrReplace

Ersetzt den angegebenen Teilstring mit einem neuen String.

StrReplace, OutputVar, Haystack, SearchText [, ReplaceText, OutputVarCount, Limit = -1]

Parameter

AusgabeVar

Name der Variable, in der das Ergebnis vom Ersetzen gespeichert werden soll.

Heuhaufen

Der String, dessen Inhalt durchsucht und ersetzt wird.

Suchtext

Der zu suchende Text. Die Übereinstimmung ist erst von der Groß- und Kleinschreibung abhängig, wenn StringCaseSense aktiv ist.

Ersatztext

Suchtext wird mit diesem Text ersetzt. Ist dieser Parameter leer oder nicht vorhanden, wird Suchtext mit einem leeren String ersetzt. Das heißt, dass der gesuchte Text in AusgabeVar wegfällt.

AusgabeVarAnzahl

Name der Variable ohne Anführungszeichen, in der die Anzahl gespeichert werden soll, wie oft der Heuhaufen ersetzt wurde (0, wenn nicht).

Limit

Ist Limit nicht vorhanden, wird standardmäßig -1 verwendet, um alle gefundenen Vorkommen des Musters im Heuhaufen zu ersetzen. Ansonsten kann angegeben werden, wie oft der Heuhaufen ersetzt werden soll. Der Bereich des Heuhaufens rechts neben dem letzten Ersatz bleibt unverändert.

Bemerkungen

Die internen Variablen A_Space und A_Tab enthalten jeweils ein einzelnes Leerzeichen und ein einzelnes Tabulatorzeichen. Sie sind nützlich, um nach Leer- und Tabulatorzeichen zu suchen, die entweder einzeln oder am Anfang oder Ende von Suchtext vorkommen.

Siehe auch

RegExReplace(), StringCaseSense, InStr, SubStr, StrLen, StrLower, StrUpper

Beispiele

; Remove all CR+LF's from the clipboard contents:
StrReplace, clipboard, clipboard, `r`n, , All

; Replace all spaces with pluses:
StrReplace, NewStr, OldStr, %A_SPACE%, +, All

; Remove all blank lines from the text in a variable:
Loop
{
    StrReplace, MyString, MyString, `r`n`r`n, `r`n, UseErrorLevel
    if ErrorLevel = 0  ; No more replacements needed.
        break
}
