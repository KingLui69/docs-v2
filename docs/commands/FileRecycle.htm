FileRecycle, Dateimuster

Parameter

Dateimuster

Name einer Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. FilePattern is assumed to be in %A_WorkingDir% if an absolute path isn't specified.

Um ein ganzes Verzeichnis zu recyclen, muss dessen Namen ohne nachfolgenden Backslash angegeben werden.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Dieser Befehl verwendet die systeminterne Funktion SHFileOperation. This function may permanently delete the file if it is too large to be recycled; also, a warning should be shown before this occurs.

Siehe auch

Beispiel

FileRecycle, C:\Temp-Dateien\*.tmp
