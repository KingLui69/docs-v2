DateDiff

Vergleicht zwei Datum-Zeit-Werte und gibt die Differenz zurück.

AusgabeVar := DateDiff(DatumZeit1, DatumZeit2, Zeiteinheit)

Parameter

DatumZeit1,
DatumZeit2

Zeitstempel im YYYYMMDDHH24MISS-Format.

Liegt DatumZeit1 zeitlich hinter DatumZeit2, ist das Ergebnis eine negative Zahl.

Zeiteinheit
Einheiten, um die Differenz zu messen. Zeiteinheit kann einer der folgenden Strings sein (oder einfach nur der erste Buchstabe):
Seconds, Minutes, Hours oder Days.

Rückgabewert

Enthält DatumZeit einen ungültigen Zeitstempel oder ein Jahr vor 1601, wird ein leerer String zurückgegeben, um das Problem zu kennzeichnen.

Ansonsten ist der Rückgabewert die Differenz zwischen zwei Zeitstempeln in der angegebenen Zeiteinheit. Das Ergebnis wird immer auf die nächste ganze Zahl abgerundet. Beträgt die Differenz zwischen zwei Zeitstempeln beispielsweise 1.999 Tage, wäre das gemeldete Ergebnis 1 Tag. Benötigt man eine höhere Genauigkeit, sollte man Seconds in Zeiteinheit angeben und das Ergebnis mit 60.0, 3600.0 oder 86400.0 dividieren.

Bemerkungen

Die integrierte Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Mit der A_TickCount-Methode kann man genau bestimmen, wieviel Zeit zwischen zwei Ereignissen verstrichen ist, weil sie eine Präzision im Millisekundenbereich ermöglicht.

Mit DateAdd können Sekunden, Minuten, Stunden oder Tage mit einem Zeitstempel addiert oder subtrahiert werden (mit einer negativen Zahl wird eine Subtraktion erreicht).

Siehe auch

DateAdd, FileGetTime, FormatTime

Beispiel

var1 := "20050126"
var2 := "20040126"
diff := DateDiff(var1, var2, "days")
MsgBox %diff%  ; Das Ergebnis ist 366, weil 2004 ein Schaltjahr ist.
