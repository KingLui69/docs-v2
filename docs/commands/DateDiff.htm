DateDiff

Compares two date-time values and returns the difference.

OutputVar := DateDiff(DateTime1, DateTime2, TimeUnits)

Parameter

DateTime1,
DateTime2

Date-time stamps in the YYYYMMDDHH24MISS format.

If DateTime1 is earlier than DateTime2, the result is a negative number.

Zeiteinheit
Units to measure the difference in. TimeUnits may be one of the following strings (or just the first letter):
Seconds, Minutes, Hours or Days.

Rückgabewert

If DateTime contains an invalid timestamp or a year prior to 1601, an empty string is returned to indicate the error.

Otherwise, the return value is the difference between the two timestamps, in the units specified by TimeUnits. The result is always rounded down to the nearest integer. Beträgt die Differenz zwischen zwei Zeitstempeln beispielsweise 1.999 Tage, wäre das gemeldete Ergebnis 1 Tag. If higher precision is needed, specify Seconds for TimeUnits and divide the result by 60.0, 3600.0, or 86400.0.

Bemerkungen

Die integrierte Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Mit der A_TickCount-Methode kann man genau bestimmen, wieviel Zeit zwischen zwei Ereignissen verstrichen ist, weil sie eine Präzision im Millisekundenbereich ermöglicht.

To add or subtract a certain number of seconds, minutes, hours, or days from a timestamp, use DateAdd (subtraction is achieved by adding a negative number).

Siehe auch

DateAdd, FileGetTime, FormatTime

Beispiel

var1 := "20050126"
var2 := "20040126"
diff := DateDiff(var1, var2, "days")
MsgBox %diff%  ; The answer will be 366 since 2004 is a leap year.
