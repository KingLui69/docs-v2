DirCopy

Kopiert einen Ordner, einschließlich seine Unterordner und Dateien (ähnlich wie xcopy).

DirCopy, Source, Dest [, Flag]

Parameter

Quelle

Name des Quellverzeichnisses (ohne Backslash), das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Zum Beispiel: C:\Mein Ordner

Ziel

Name des Zielverzeichnisses (ohne Backslash danach), das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Zum Beispiel: C:\Kopie von Mein Ordner

Flag

(Optional) Dieser Parameter bestimmt, ob bereits vorhandene Dateien überschrieben werden dürfen:

0 (Standard): Vorhandene Dateien werden nicht überschrieben. Die Operation wird fehlschlagen und keinen Effekt haben, wenn Ziel bereits als Datei oder Verzeichnis existiert.

1: Vorhandene Dateien werden überschrieben. However, any files or subfolders inside Dest that do not have a counterpart in Source will not be deleted.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt. Enthält das Quellverzeichnis allerdings eine gespeicherte Webseite, bestehend aus Seitenname.htm und dem dazugehörigen Verzeichnis Seitenname_Dateien, erzeugt dieser Befehl möglicherweise ein Fehler, auch wenn das Kopieren erfolgreich war.

Bemerkungen

Wenn die Verzeichnisstruktur des Ziels nicht vorhanden ist, wird sie automatisch erstellt, falls möglich.

Da die Operation in der Lage ist, einen Ordner mit seinen Unterordnern und Dateien rekursiv zu kopieren, ist das Ergebnis beim Kopieren eines Ordners auf ein Ziel irgendwo in sich selbst nicht definiert. To work around this, first copy it to a destination outside itself, then use DirMove to move that copy to the desired location.

DirCopy copies a single folder. Um stattdessen die Inhalte eines Ordners zu kopieren (einschließlich seine Dateien und Unterordner), siehe Beispiele bei FileCopy.
