If (Ausdruck)

Ein oder mehrere Befehle, die ausgeführt werden, wenn ein Ausdruck als WAHR ausgewertet wird. 

If (Ausdruck)

Bemerkungen

Falls der Ausdruck der If-Anweisung als wahr gewertet wird (das heißt, dass das Ergebnis weder leer noch eine 0 sein darf), wird die Zeile oder der Block darunter ausgeführt. Ansonsten wird die Zeile oder der Block beim nachfolgenden ELSE ausgeführt, falls vorhanden.

Sobald IF oder ELSE mehrere Zeilen enthält, müssen diese mit Klammern umschlossen werden. Wenn IF oder ELSE nur eine Zeile ausführt, können die Klammern weggelassen werden. Entsprechende Beispiele können im untersten Abschnitt gefunden werden.

The One True Brace (OTB) style may optionally be used. Zum Beispiel:

If (x < y) {
    ...
}
If WinExist("Unbenannt - Editor") {
    WinActivate
}
If IsDone {
    ...
} Else {
    ...
}

Any type of statement can appear immediately to the right of an if-statement, but unlike an else-statement, a delimiting comma is required. Zum Beispiel:

if true, MsgBox  ; Good
if true  MsgBox  ; Bad

Siehe auch

Expressions, Assign expression (:=), Ternary operator (a?b:c), Blocks, Else, While-loop

Beispiel

If (A_Index > 100 or Fertig)
    Return

If (A_TickCount - Startzeit > 2*Maximalzeit + 100)
{
    MsgBox Zu viel Zeit verstrichen.
    ExitApp
}

If (Farbe = "Blau" or Farbe = "Weiß")
{
    MsgBox Die Farbe ist einer der erlaubten Werte.
    ExitApp
}
Else If (Farbe= "Silber")
{
    MsgBox Silber ist keine erlaubte Farbe.
    Return
}
Else
{
    MsgBox Diese Farbe wurde nicht erkannt.
    ExitApp
}
