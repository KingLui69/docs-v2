DateAdd

Addiert oder Subtrahiert Zeit von einem Datum-Zeit-Wert.

AusgabeVar := DateAdd(DatumZeit, Zeit, Zeiteinheit)

Parameter

DatumZeit

Ein Zeitstempel im YYYYMMDDHH24MISS-Format.

Zeit

Die Zeitmenge, die addiert werden soll (Integer oder Fließkommazahl). Mit einer negativen Zahl kann eine Subtraktion durchgefüht werden.

Zeiteinheit
Die Bedeutung des Zeit-Parameters. Zeiteinheit kann einer der folgenden Strings sein (oder einfach nur der erste Buchstabe):
Seconds, Minutes, Hours oder Days.

Rückgabewert

Enthält DatumZeit einen ungültigen Zeitstempel oder ein Jahr vor 1601, oder ist Zeit nicht-numerisch, wird ein leerer String zurückgegeben, um das Problem zu kennzeichnen.

Ansonsten wird ein Zeitstempel im YYYYMMDDHH24MISS-Format zurückgegeben.

Bemerkungen

Die integrierte Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Mit DateDiff kann die Zeitmenge zwischen zwei Zeitstempeln berechnet werden.

Siehe auch

DateDiff, FileGetTime, FormatTime

Beispiel

; Addiert das heutige Datum um 31 Tage.
später := DateAdd(A_Now, 31, "days")
MsgBox % FormatTime(später)
