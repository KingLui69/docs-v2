EnvUpdate

Benachrichtigt das Betriebssystem und alle laufenden Anwendungen, dass die Umgebungsvariablen geändert worden sind.

EnvUpdate

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Aktualisiert die OS-Umgebung. Das ist das gleiche wie, als würde man sich ab- und wieder anmelden.

EnvUpdate kann beispielsweise dazu verwendet werden, mögliche Änderungen am folgenden Registry-Schlüssels bekannt zu machen: HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Session Manager\Environment

Siehe auch

EnvSet, RegWrite
