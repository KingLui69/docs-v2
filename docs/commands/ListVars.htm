ListVars

Displays the script's variables: their names and current contents.

ListVars

Bemerkungen

Dieser Befehl ist das gleiche wie, als würde man den Menüpunkt "View->Variables" im Hauptfenster auswählen. It can help you debug a script.

Jede Zeile in der Liste beinhaltet:
1) Den Namen der Variable.
2) Die Länge des Variableninhalts und die Variablenkapazität. Zum Beispiel: [50 of 63]
3) Die ersten 60 Zeichen des Variableninhalts

Befindet sich dieser Befehl innerhalb einer Funktion, werden zuerst die lokalen Variablen der Funktion aufgelistet (über den globalen Variablen des Scripts).

Bekannte Einschränkung: Enthält eine Funktion (oder die Liste der globalen Variablen selbst) mehr als 10.000 Variablen, besteht die Möglichkeit, dass dieser Befehl die Variablen nicht genau in alphabetischer Reihenfolge anzeigen wird; das könnte dazu führen, dass einige Variablen in der Liste fehlen.

Beispiel

Function Syntax

var1 := "foo"
var2 := "bar"
obj := []
ListVars()
Pause()

Command Syntax

var1 := "foo"
var2 := "bar"
obj := []
ListVars
Pause
