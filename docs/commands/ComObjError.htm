ComObjError()

Aktiviert oder deaktiviert COM-Fehlermeldungen.

Aktiviert := ComObjError([Aktivieren])

Parameter

Aktivieren

Ein boolescher Wert (True oder False). Optional.

Aktiviert

Receives the setting which was in effect before the function was called.

Allgemeine Bemerkungen

Nach dem Zugriff auf ein COM-Objekt enthält A_LastError den HRESULT-Code, der von der IDispatch::Invoke-Funktion des COM-Objekts zurückgegeben wurde. Scripts können ihre eigene Fehlerbehandlung durch den Aufruf von ComObjError(false) implementieren, um Fehlermeldungen zu deaktivieren und A_LastError hinzuziehen.
