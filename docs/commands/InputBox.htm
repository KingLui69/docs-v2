InputBox

Zeigt ein Fenster zur Eingabe eines Strings an.

InputBox, OutputVar [, Title, Prompt, Options, Default]

Parameter

AusgabeVar

Name der Variable, in der die Benutzereingabe gespeichert werden soll.

Titel

Der Titel der InputBox. Ist dieser Parameter leer oder nicht vorhanden, wird standardmäßig der aktuelle Name des Scripts verwendet.

Anzeigetext

Text des Eingabefensters, der darauf hinweist, was eingegeben werden soll. Falls der Anzeigetext zu lang ist, kann dieser in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um die Lesbarkeit und Wartbarkeit zu verbessern.

Optionen

A string of case-insensitive options, with each separated from the last by a space or tab.

Xn Yn: The X and Y coordinates of the dialog. For example, X0 Y0 puts the window at the upper left corner of the desktop. If either coordinate is omitted, the dialog will be centered in that dimension. Either coordinate can be negative to position the dialog partially or entirely off the desktop (or on a secondary monitor in a multi-monitor setup).

Wn Hn: The width and height of the dialog. For example, W375 H189 is the default size.

T: Specifies the timeout in seconds. For example, T10.0 is ten seconds. Der Parameter wird automatisch auf 2147483 (24.8 Tage) zurückgesetzt, falls dieser Wert überschritten wird. After the timeout has elapsed, the InputBox window will be automatically closed and ErrorLevel will be set to 2. OutputVar will still be set to what the user entered.

Password: Mask the user's input. To specify which character is used, follow this example: Password*

Standardwert

Ein String, der anfänglich im Eingabefeld des Dialogfensters angezeigt wird. Der Benutzer kann den String z. B. durch Drücken der Rücktaste ändern.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

Bemerkungen

Mit diesem Dialogfenster kann der Benutzer einen Text eingeben und danach OK oder CANCEL (Abbrechen) drücken. Durch Ziehen der Rahmen kann dessen Größe geändert werden.

ErrorLevel wird auf 1 gesetzt, wenn der Benutzer die Schaltfläche CANCEL drückt, auf 0, wenn der Benutzer OK drückt, oder auf 2, wenn das Dialogfenster die Zeit überschritten hat. In allen drei Fällen wird AusgabeVar auf den eingegebenen Wert gesetzt. Dadurch kann die Schaltfläche CANCEL außer Abbrechen noch andere Funktionen ausführen, falls der Entwickler das wünscht.

Ein GUI-Fenster kann eine modale InputBox mittels Gui, +OwnDialogs anzeigen. Eine modale InputBox hindert den Benutzer daran, mit dem GUI-Fenster zu interagieren, bis die InputBox wieder geschlossen wird.

Beispiel

InputBox, password, Enter Password, (your input will be hidden), password 
InputBox, UserInput, Phone Number, Please enter a phone number., w640 h480
if ErrorLevel
    MsgBox, CANCEL was pressed.
else
    MsgBox, You entered "%UserInput%"
