WinKill

Zwingt das angegebene Fenster sich zu schließen.

WinKill [, Fenstertitel, Fenstertext, Wartezeit, Titelausnahme, Textausnahme]

Parameter

Fenstertitel

Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.

Fenstertext

In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (abrufbar mit Windows Spy). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.

Wartezeit

Ist dieser Parameter leer oder nicht vorhanden, wartet der Befehl überhaupt nicht. Ist eine 0 angegeben, wartet der Befehl 500 ms. Ansonsten wartet der Befehl die angegebene Anzahl an Sekunden (kann einen Dezimalpunkt oder einen Ausdruck enthalten), bis das Fenster geschlossen wird. Falls das Fenster innerhalb der Wartezeit nicht geschlossen wird, setzt das Script seine Ausführung fort. ErrorLevel is not set by this command, so use WinExist or WinWaitClose if you need to determine for certain that a window is closed.

Titelausnahme

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Zuerst versucht dieser Befehl, das Fenster auf normale Weise zu schließen. Funktioniert das nicht, wird der Befehl versuchen, das Fenster durch Beenden seines Prozesses zu schließen.

Ist ein übereinstimmendes Fenster aktiv, wird vorrangig dieses Fenster geschlossen, anstatt ein anderes übereinstimmende Fenster darunter. Im Falle mehrerer übereinstimmender Fenster wird generell das oberste (aktuell verwendete) Fenster geschlossen.

Dieser Befehl bezieht sich immer auf das oberste übereinstimmende Fenster. Enthält Fenstertitel allerdings ahk_group Gruppenname, sind alle Fenster von der Gruppe betroffen.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

If WinExist("Untitled - Notepad")
    WinKill ; use the window found above
