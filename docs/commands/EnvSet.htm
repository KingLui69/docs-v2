EnvSet

Schreibt einen Wert eine Variable, die in der Systemumgebung enthalten ist.

EnvSet, UmgebungsVar, Wert

Parameter

UmgebungsVar
Name der Umgebungsvariable, z. B. "COMSPEC" oder "PATH".
Wert

Wert, der in die Umgebungsvariable gespeichert werden soll.

ErrorLevel

This command is able to throw an exception on failure. For more information, see Runtime Errors.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB.

Umgebungsvariablen, die mit diesem Befehl erstellt oder geändert wurden, können nur bei Programmen verwendet werden, die ein Script per Run oder RunWait gestartet hat. Siehe Umgebungsvariablen für mehr Details.

This command exists because normal script variables are not stored in the environment. Zum einen, weil das die Performance negativ beeinträchtigen würde, um zum anderen, weil das Betriebssystem jede Umgebungsvariable auf 32 KB begrenzt.
