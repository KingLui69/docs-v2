ExitApp

Beendet das Script bedingungslos.

ExitApp [, ExitCode]

Parameter

ExitCode

Ein Integer-Wert zwischen -2147483648 und 2147483647 (kann ein Ausdruck sein), welcher zurückgegeben wird, wenn das Script beendet wird. Dieser Code ist für jedes Programm zugänglich, welches das Script startet, wie beispielsweise durch ein anderes Script (mittels RunWait) oder einer Batch-Datei (.bat). Sobald ExitCode weggelassen wird, gilt standardmäßig Null. Eine Null wird normalerweise verwendet, um Erfolg zu zeigen.

Bemerkungen

Das Script wird sofort beendet, sofern es keine OnExit-Subroutine hat. Das ist das gleiche wie, als würde "Exit" im Tray-Menü des Scripts oder im Hauptmenü ausgewählt.

ExitApp is often unnecessary in scripts which are not persistent.

Hat das Script eine OnExist-Subroutine, wird es als Reaktion auf ExitApp ausgeführt.

Siehe auch

Exit, OnExit, #Persistent

Beispiel

#x::ExitApp  ; Mit diesem Hotkey kann das Script beendet werden.
